# Data Marketplace API

This projects holds the classes, interfaces 
other abstractions of the TAUS Data Marketplace Product.

## Development Setup

Requirements
1. JDK 13

All below commands are expected to be executed from the project root directory.

**Build**

```shell script
./gradlew build
```

**Test**

```shell script
./gradlew test
```

**Run**

```shell script
./gradlew bootRun
```
